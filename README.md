**Course:** INS 322 – Web Design and Management  
**Assignment:** CAT I  
**Student Name:** Dorcas Lotodo

## Project Description

This project is a **three-page Curriculum Vitae (CV) website** created as part of the **INS 322 Web Design and Management CAT I assignment**.

The website presents **personal, academic, and professional information about Dorcas Lotodo** in a clean and accessible format using **HTML only** (no CSS stylesheets).

## Page Descriptions

### 1. Home Page (`index.html`)
- Navigation bar linking all three pages
- Profile photo and introduction
- Personal summary / About Me section
- Education history displayed in a table

---

### 2. Experience & Skills (`experience.html`)
- Navigation bar linking all three pages
- Work and university experience presented in a table
- List of professional skills
- Hobbies and interests section

---

### 3. Contact Page (`contact.html`)
- Navigation bar linking all three pages
- Contact details (name, email, location) shown in a table
- References table with referee names and emails

## How to Run the Website

1. Ensure all files are placed in their correct folders as shown in the **folder structure**.
2. Open `index.html` in any modern web browser such as:
   - Google Chrome
   - Mozilla Firefox
   - Microsoft Edge
3. Use the **navigation bar** at the top of each page to move between sections of the website.

> No internet connection is required since all files are stored locally.
